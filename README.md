# gwfm
Fit Generalized Win Fraction Regression Models

Fit a generalized win-fraction regression as described in Cao, Fang and Li (2026) Generalized win fraction regression for composite survival endpoints, which can obtain parameter coefficients estimation, corresponding standard errors as well as P value.

The manual of gwfm package for the manuscript ``Cao, Fang and Li (2026) Generalized win fraction regression for composite survival endpoints" is also provided.

The simulation_code folder includes all R code to repeoduce simulation study results in our paper “Generalized win fraction regression for composite survival endpoints"
